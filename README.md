# Sticky notes

## Usage

### Keybinds

| KeyStroke    | Action               |
|--------------|----------------------|
| CTRL + W     | Close focused window |
| CTRL + T     | Create new note      |
| CTRL + D     | Delete last note     |
| CTRL + ENTER | Enter last note      |
| 1-9          | Open [1-9]th note    |  

### Directories created
```
    home_dir:.
    ├───.config
    └───notes
```
.config contains the index file of the note files.
If deleted bad things will happen.<br>
notes contains the actual note files nested in a folder named with the file name.
